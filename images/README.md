# Images Directory

This directory is for storing all images and media assets for the cha-ching website.

## Usage

Place your image files here and reference them in your HTML files using relative paths:

```html
<img src="images/your-image.jpg" alt="Description">
```

## Supported Formats

- `.jpg` / `.jpeg` - Photographs and complex images
- `.png` - Images with transparency
- `.gif` - Animated images
- `.svg` - Scalable vector graphics
- `.webp` - Modern image format for better compression

## Organization Tips

Consider organizing images into subdirectories:

```
images/
├── gallery/
├── backgrounds/
├── icons/
├── banners/
└── thumbnails/
```

## File Naming

Use descriptive, lowercase filenames with hyphens:
- ✅ Good: `profile-photo.jpg`, `banner-header.png`
- ❌ Avoid: `Image1.jpg`, `PHOTO_2024.png`
